# 设计师个人网站

一个现代简约风格的设计师个人网站，采用蓝色主题色，整体使用圆角、毛玻璃、卡片布局设计，完美适配 PC 和移动端。

## 🎨 设计特点

- **主题色**：蓝色 + 淡紫色渐变
- **设计风格**：现代简约 × 清新二次元
- **视觉效果**：圆角卡片、毛玻璃背景、悬浮阴影
- **配色方案**：浅白 + 淡粉紫渐变低饱和配色，治愈柔和
- **响应式设计**：适配 1920x1080 桌面端及移动端设备

## 📄 页面模块

### 5. 联系方式 (index.html#contact)

## 🚀 使用说明

### 本地预览

由于环境限制，您可以直接双击打开 `index.html` 文件在浏览器中预览，或者使用以下任一方式：

**方式 1：使用浏览器直接打开**
```
直接双击 index.html 文件
```

**方式 2：使用 Python 启动服务器**
```bash
cd e:\新建文件夹 (2)\222
python -m http.server 8888
```
然后在浏览器访问：http://localhost:8888

**方式 3：使用 Node.js 启动服务器**
```bash
cd e:\新建文件夹 (2)\222
npx http-server -p 8888
```
然后在浏览器访问：http://localhost:8888

### 自定义图片

网站目前使用 picsum.photos 的在线图片服务作为占位符。要使用自己的图片：

1. 将图片放入 `images` 文件夹
2. 修改 `index.html` 中的图片路径

**替换示例：**
```html
<!-- 替换前 -->
<img src="https://picsum.photos/seed/avatar/600/800" alt="个人形象">

<!-- 替换后 -->
<img src="images/avatar.png" alt="个人形象">
```

**需要的图片：**
- `avatar.png` - 个人头像/IP 形象（600x800px）
- `work1.png` ~ `work6.png` - 作品展示图（380x220px）
- `wechat-qr.png` - 微信二维码（160x160px）
- `qq-qr.png` - QQ 二维码（160x160px）
